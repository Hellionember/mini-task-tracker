# Mini Task Tracker 🚀

Мини-трекер задач с продвинутой логикой статусов, умной серверной сортировкой и реактивным интерфейсом. Выполнены **все базовые, нетривиальные и бонусные требования**!

## 🛠 Стек технологий
* **Backend:** Python 3.11+, FastAPI, Pydantic (мгновенная жесткая валидация)
* **Database:** SQLite (данные сохраняются между перезапусками, БД создается автоматически)
* **Frontend:** HTML, CSS, Alpine.js (реактивность без сложной сборки и Node.js)

---

## 🚀 Как запустить (Одной командой)

Убедитесь, что у вас установлен Python.

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите сервер:
```bash
uvicorn main:app --reload
```
Приложение будет доступно по адресу: **http://127.0.0.1:8000**
