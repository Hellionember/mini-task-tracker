# Mini Task Tracker

Мини-трекер задач с продвинутой логикой статусов, умной сортировкой и реактивным интерфейсом. 

## Стек технологий
* **Backend:** Python, FastAPI, Pydantic (мгновенная валидация данных)
* **Database:** SQLite (встроенная, данные сохраняются между перезапусками)
* **Frontend:** HTML, CSS (Tailwind-подобный custom), Alpine.js (реактивность без сборки)

## Запуск приложения (Одной командой)

Убедитесь, что у вас установлен Python и зависимости (`fastapi`, `uvicorn`, `pydantic`).

Выполните в корневой папке проекта:
```bash
uvicorn main:app --reload
